AGENT SDK
4
Claude Reasoning Loop (up to 25 turns)
@anthropic-ai/claude-agent-sdk
Claude reads the system prompt + user question, then decides which tools to call.

For "Which regions have highest complaint volume?" a typical loop:
  1. Calls search_graph — finds complaint-related tables/SPs
  2. Calls query_db — joins ComplaintTRN → CityMaster → DistrictMaster → StateMaster
  3. Gets rows back, formats the final answer with table + analysis

Each tool call goes to the MCP Server below. The SDK handles the tool_use → tool_result → next_turn cycle automatically.
